# 💬 10 Numara

> A real-time live-chat utility built with Flutter & Firebase.

![Flutter](https://img.shields.io/badge/Flutter-02569B?logo=flutter&logoColor=white)
![Firebase](https://img.shields.io/badge/Firebase-FFCA28?logo=firebase&logoColor=black)

## About

**10 Numara** is a real-time messaging app built on a Firebase backend, featuring live chat and asynchronous data synchronization — a fast communication tool that doesn't require contact/phonebook integration.

## Features

- 💬 Real-time live chat (Firebase)
- 🔄 Asynchronous data synchronization
- ⚡ Optimized database query logic
- 📱 Clean, responsive UI

## Tech stack

- **Flutter** (Dart)
- **Firebase** (Firestore / Auth)

## Run locally

```bash
git clone https://github.com/ismetguler/10Numara.git
cd 10Numara
flutter pub get
flutter run
```
